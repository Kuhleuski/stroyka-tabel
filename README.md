# 🏗️ Стройка Табель 

Приложение для учёта рабочего времени строительной бригады. Позволяет отслеживать, кто из рабочих и на каких объектах работал в конкретные дни.

## 📱 Функционал

### Основные возможности

- **Два режима просмотра:**
  - **Месяц** — классический календарь с отметками о рабочих днях
  - **День** — бесконечная лента дней с детальной информацией по каждому дню

- **Детальный просмотр дня** — клик по любому дню открывает полную информацию:
  - Какие объекты работали
  - Какие рабочие были на каждом объекте
  - Количество отработанных часов

- **Умное сохранение позиции** — при закрытии детального просмотра лента возвращается ровно на то место, где вы были

- **Актуальная дата по центру** — при первом переходе на вкладку "День" сегодняшняя дата показывается по центру экрана

### Страницы

| Страница | Описание |
|----------|----------|
| **Главная** | Календарь (месяц/день) + информация за выбранный день |
| **Объекты** | Список всех объектов с количеством смен и часов |
| **Бригада** | Список всех рабочих с общей статистикой |

## 🗄️ База данных (Supabase)

### Таблица `shifts`

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | BIGSERIAL | Первичный ключ |
| `worker_name` | TEXT | Имя рабочего |
| `site_name` | TEXT | Название объекта |
| `work_date` | DATE | Дата работы |
| `hours` | DECIMAL | Количество часов |
| `status` | TEXT | `confirmed` или `pending` |
| `created_at` | TIMESTAMP | Дата создания записи |

### Пример данных

```sql
